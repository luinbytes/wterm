Why wterm?

Because you deserve a terminal that's as fast and modern as your workflow.

🎨

Warp-Inspired UI

Beautiful dark theme with Tokyo Night colors. Command blocks that actually make sense. Your eyes will thank you.

⚡

Blazing Fast

Built in Go with Bubble Tea. Single binary, zero runtime dependencies. Starts instantly, runs smoothly.

🤖

AI-Ready

Bring your own API key. AI integration stubbed and ready. Type /ai and get explanations, commands, whatever.

📦

Command Blocks

Each command + output grouped together. No more scrolling through endless noise. Clean. Organized.

📝

Persistent History

Never lose a command again. Configurable history with search. Your muscle memory, preserved.

🔧

Configurable

Override colors, set history limits, add your API key. YAML config at ~/.warp-clone/config.yaml
